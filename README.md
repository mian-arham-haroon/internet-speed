# 🌐 Internet Speed Tester

A clean Python desktop app to check your internet download and upload speed with a simple Tkinter interface.

## ✨ Highlights

- ⚡ Fast download test
- 📤 Fast upload test
- 🖥️ Easy-to-use GUI
- 🔧 Built with Python
